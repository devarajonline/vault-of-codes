# 🔐 Secret Code Generator

A simple Python-based Secret Code Generator that uses the Caesar Cipher technique to encode and decode messages. This project allows users to convert plain text into secret coded text and decode it back using a shift value.

## 📌 Features

- Encode messages using a shift value
- Decode coded messages back to original text
- Supports uppercase and lowercase letters
- Preserves spaces, numbers, and special characters
- Menu-driven command-line interface
- Handles alphabet wrap-around (e.g., Z → B)

## 🛠️ Technologies Used

- Python 3
- Functions
- Loops
- String Manipulation
- Conditional Statements

## 🚀 How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open a terminal in the project directory.
4. Run the program:

```bash
python secret_code_generator.py
```

## 📋 Menu Options

```text
1. Encode Message
2. Decode Message
3. Exit
```

## 💡 Example

### Encoding

```text
Enter your choice: 1
Enter message: Hello World
Enter shift value: 3

Encoded Message: Khoor Zruog
```

### Decoding

```text
Enter your choice: 2
Enter coded message: Khoor Zruog
Enter shift value: 3

Decoded Message: Hello World
```
